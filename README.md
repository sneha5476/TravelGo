# 🌍 TravelGo – Travel Booking Web Application

TravelGo is a simple and responsive travel booking web application that allows users to explore destinations, book travel packages, and manage their bookings efficiently. The project is built using core frontend technologies with a focus on interactivity and user experience.

---

## 🚀 Features

- 🏠 Multi-page navigation (Home, Packages, Booking, About, Contact)
- ✈️ Dynamic travel package display
- 🧾 Booking system with form validation
- 💾 LocalStorage-based data storage
- ➕ Add and ❌ Delete bookings (CRUD operations)
- 🎯 Auto-fill destination from selected package
- 📩 Contact form with email validation
- 📱 Responsive and clean UI design

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript 
- LocalStorage API
